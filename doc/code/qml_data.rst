qml.data
=========

Overview
--------

The data module provides functionality to access, store and manipulate quantum datasets within the PennyLane framework.

The functionality contained in this module can be used to manage datasets using the :class: `~pennylane.data.Dataset` class.
Pre-computed datasets are available for download and can be accessed using the :func: `~pennylane.data.load` function.
Alternatively, users can easily create, write to disk, and read custom datasets using the :class: `~pennylane.data.Dataset` class.
